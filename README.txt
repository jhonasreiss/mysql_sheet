Software focado em inserção rápida de transferência de dados de planilhas no banco de dados.
Possui 2 opções de inserção:
-xlsx
-csv

A interface do software lhe permite ver as colunas disponíveis do arquivo carregado
e a possibilidade de escolher qual coluna deseja inserir os dados no banco de dados.

Também permite coletar dados de banco de dados SQL para exportar em planilha (xlsx / csv)
